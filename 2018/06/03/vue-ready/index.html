some every includes find

  • 上面的例子中，都是查找是否含有‘5’
  • 通过上面的例子，可以发现都是有‘查找’的功能，特点如下：
  • 1、includes，返回 boolean，即找到返回 true ，没有找到返回false
  • 2、find，当找到的值，停止循环，并将值返回
  • 3、some，找 true，找到 true 后停止 返回 true 找不到返回false
  • 4、every，找 false，找到 false 后停止，返回 false
