（9）储存

发表于 2018-07-27 | 分类于 面试题

面试的题目

请描述一下 cookie, sessionStorage 和 localStorage 的区别?

考察的知识点

cookie

本身是用于客户端和服务端通信,但是它有本地存储的功能,于是就被借用;
使用document.cookie = … 获取和修改即可。

cookie具有以下特点：

储存容量太小, 只有 4KB
所有 http 请求都会带着cookie,会影响获取资源的效率
API 简单,需要封装才能用 document.cookie
不设置的情况下，关闭浏览器，浏览器cookie消失

sessionStorage

sessionStorage与cookie比较一致。

sessionStorage特点：

理论值可以不限
一般用于会话

localStorage

localStorage的特点

储存容量, 最大5M
不清除，可以永久保存
API 简单,localStorage.getItem(key), localStorage.setItem(key, value)
